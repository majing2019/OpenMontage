# Seedream Prompt Patterns (归藏提示词库)

## How to Use

These examples are NOT templates to copy-paste. They are **pattern references** —
study how they structure prompts and apply the same patterns to healing-text scenes.

## Key Patterns Observed

1. **Structured sections**: 画面描述 → 细节 → 氛围与光影 → 参数设置
2. **Concrete over abstract**: "清晨五点半的黄山" not "mountain at dawn"
3. **Chinese-native**: All descriptions in natural Chinese, not translated photography terms
4. **Layered detail**: Start broad (scene), then narrow (specific elements), then atmosphere, then technical
5. **Negative space awareness**: Always specify where text/composition breathing room should be

## For Healing-Text Specifically

When adapting these patterns for healing-text landscape scenes:

- Replace "产品" (product) focus with "自然" (nature) focus
- Keep the structured layering: 场景 → 光影 → 细节 → 参数
- Maintain the Chinese-native description style
- Add "画面上方/中央留白用于文字叠加" to every prompt
- Use the level of concrete detail these examples demonstrate — time of day, weather, season, specific location names
