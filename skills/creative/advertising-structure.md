# Advertising Structure for Short-Form Video

> Sources: Luke Sullivan "Hey, Whipple, Squeeze This" (1998),
> 《商业短片广告创作》, 《广告师手册》, 《网店商品视频拍摄与制作》

## AIDA Model (Core Framework)

The foundational advertising structure for conversion-driven short videos.

| Stage | Timing | Goal | Content |
|-------|--------|------|---------|
| **Attention** | 0-3s | Stop the scroll | Visual shock, bold text, pattern interrupt |
| **Interest** | 3-10s | Why should I care? | Problem statement, pain point, relatable scenario |
| **Desire** | 10-45s | I want this solution | Product/service introduction, benefit demonstration |
| **Action** | 45-60s | What do I do next? | CTA: follow, buy, share, comment |

### AIDA Timing for Different Durations

| Duration | Attention | Interest | Desire | Action |
|----------|-----------|----------|--------|--------|
| 15s | 0-2s | 2-5s | 5-12s | 12-15s |
| 30s | 0-3s | 3-8s | 8-25s | 25-30s |
| 60s | 0-3s | 3-15s | 15-52s | 52-60s |

## The Golden 3-Second Rule

**Users decide to stay or scroll in 3 seconds.** This is the only "interview" your video gets.

### 8 Hook Strategies for the First 3 Seconds

| Strategy | Chinese | Template | Example |
|----------|---------|---------|---------|
| Conflict-first | 冲突开场 | 从矛盾中已经开始 | "这个方法让我被骗了三年" |
| Anti-common-sense | 反常识 | 打破观众的固有认知 | "你一直做错的3件事" |
| Visual shock | 视觉冲击 | 不解释，直接震撼 | 产品从高处摔下但完好无损 |
| Direct question | 直接提问 | 抛出观众必须回答的问题 | "你知道你每天浪费了多少时间吗？" |
| Result promise | 结果承诺 | 直接展示最终效果 | "30天后从这样→变成这样" |
| Identity anchor | 身份锚定 | 告诉观众"这是给你的" | "给所有正在加班的人" |
| Suspense creation | 制造悬念 | 抛出一个悬念 | "最后这件事发生的时候，所有人都沉默了" |
| Emotional trigger | 情感触发 | 直接触发情绪 | "凌晨3点收到妈妈的消息" |

## Structure Templates

### Problem-Solution Structure

```
[0-3s]  PAIN: 展示痛点场景，让观众点头"对对对就是我"
[3-8s]  AMPLIFY: 放大痛点——这个问题有多普遍、多严重
[8-15s] SOLUTION: 引入方案/产品——简洁具体
[15-40s] PROOF: 展示效果——前后对比、数据、用户反馈
[40-50s] DETAIL: 展示独特卖点——为什么选这个不选别的
[50-60s] CTA: 明确的行动号召
```

### Story-Based Ad Structure

```
[0-3s]  HOOK: 角色在困境中——观众立即共情
[3-15s] CHARACTER: 展示角色特点——让人喜欢这个角色
[15-30s] CONFLICT: 问题加剧——观众想知道怎么解决
[30-45s] TURNING POINT: 产品/方案出现——转折点
[45-55s] RESOLUTION: 问题解决——展示美好结果
[55-60s] BRAND: 品牌连接 + CTA
```

### Suspense-Based Ad Structure

```
[0-3s]  SUSPENSE HOOK: 抛出一个让人想知道答案的问题
[3-20s] MISDIRECTION: 引导观众朝一个方向思考
[20-40s] REVEAL: 出人意料的真相或反转
[40-50s] CONNECTION: 把真相和品牌/产品连接起来
[50-60s] CTA: 行动号召
```

## Selling Point Front-Loading Principle

**Put the strongest selling point first.** Users may leave at any moment.

```
❌ 错误: 铺垫→铺垫→铺垫→卖点 (观众可能已经走了)
✅ 正确: 卖点→证明→更多细节 (观众已经得到了核心信息)
```

### Information Priority Over Aesthetics

Every frame must deliver **effective information**. Beauty is secondary to clarity.

- Can the viewer understand the key message from this frame alone?
- Is the selling point visible in the first frame?
- Are the subtitles readable without sound? (85% watch muted)

---

## Subtitle as Independent Information Layer

**85% of feed videos are watched on mute.** Subtitles must independently convey core information.

| Requirement | Value |
|-------------|-------|
| Font size | 42px+ at 1080p |
| Max chars/line | 30 |
| Max lines | 2 |
| Position | Center or lower-center, within safe zone |
| Animation | Entrance animation (scale pop, 0.2-0.3s) |
| Highlight | Word-by-word highlight recommended |

## The "1+1=3" Visual-Text Collision Principle (Sullivan)

Visual and text should **collide** (not repeat) to create a **third meaning**.

```
❌ 文字重复画面: 画面=人在跑，文字="他在跑步" (1+1=1, 浪费)
✅ 文字碰撞画面: 画面=老人在跑，文字="昨天他刚过了80岁生日" (1+1=3, 冲击感)
```

**Applications:**
- Visual shows A, text says NOT A → irony/tension
- Visual shows ordinary, text reveals extraordinary context → surprise
- Visual shows result, text shows the cost → depth

---

## Minimal Visual Principle

**Every additional element weakens the power.** If you can communicate with one image, use one image.

- One hero shot > three mediocre shots
- One bold text overlay > five bullet points
- One powerful visual metaphor > literal demonstration

## Platform-Specific Optimization

| Platform | Key Differences |
|----------|----------------|
| Douyin | 3s hook critical, vertical composition, trending BGM boosts |
| Xiaohongshu | Lifestyle aesthetic, product detail shots, "种草" tone |
| Kuaishou | Authentic/raw feel works, relatable everyday stories |
| WeChat Channels | Longer acceptable (1-3min), more information density |

### BGM Strategy

- **Hot BGM** → platform recommendation boost
- **Music as rhythm skeleton** → edit cuts on beat points
- **Match emotion** → energetic for product, gentle for lifestyle
- **Start immediately** → no silent intro

## Ad Formula for Comics (Adapted to Short-Form)

When adapting advertising structure to comic-style short videos:

1. **Hook panel**: One panel that combines visual shock + text hook
2. **Pain panels**: 2-3 panels showing the problem with relatable details
3. **Solution panel**: 1 panel introducing the solution/turning point
4. **Proof panels**: 2-3 panels showing the before/after contrast
5. **CTA panel**: Final panel with clear action prompt

**Key difference from narrative comics**: Every panel must carry **selling information**. There's no room for "atmosphere" panels in 60-second ads.
