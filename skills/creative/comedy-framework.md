# Comedy Framework for Short-Form Content

> Source: John Vorhaus "The Comic Toolbox: How to Be Funny Even If You're Not" (1994)
> Supplementary: 《喜剧工具箱》读书笔记, 《爆款短剧创作》喜剧章节

## Core Formula

**Comedy = Truth + Pain + Distance**

- **Truth**: 必须基于观众能认同的真实观察
- **Pain**: 角色必须经历某种困境或窘迫
- **Distance**: 观众需要足够的情感距离才能笑（不是我在笑，是"他在笑"）

如果只有 Truth + Pain，那是悲剧。加上 Distance，才是喜剧。

## Comic Character Formula

**Strong Desire + Huge Flaw + Complete Sincerity = Comedy Engine**

| Component | What it means | Example |
|-----------|--------------|---------|
| Strong Desire | 角色极度想要某样东西 | 妈妈想和孩子保持亲密 |
| Huge Flaw | 用完全错误的方式追求 | 用过度控制和唠叨代替倾听 |
| Complete Sincerity | 角色真心相信自己在做对的事 | 真的觉得"这都是为了你好" |

**Key insight**: 喜剧角色必须是认真的。如果角色知道自己很可笑，观众反而不觉得好笑了。

### 角色原型示例

| 原型 | Desire | Flaw | Why Funny |
|------|--------|------|----------|
| 控制狂妈妈 | 保持家庭完整 | 用控制代替爱 | 认真得令人窒息 |
| 假装精英打工人 | 被人认可 | 全是装的 | 随时可能崩塌的体面 |
| 强装淡定的人 | 保持形象 | 内心慌得一批 | 努力维持但总露馅 |
| 过度认真的人 | 把小事做到极致 | 分不清主次 | 把所有事都当生死大事 |

## Comedy Techniques

### 1. Fish Out of Water

把一个不属于某个环境的人放进那个环境——碰撞产生喜剧。

**Formula**: 错误的人 + 错误的环境 = 自然的喜剧

| Example | Setup |
|---------|-------|
| 严厉老板去幼儿园开家长会 | 控制欲在儿童世界完全失效 |
| 外卖小哥意外进入米其林餐厅 | 两个世界的碰撞 |
| 爸爸独自带娃一整天 | "不会带孩子"的真相暴露 |

**关键**: 角色必须**认真尝试融入**——越努力越搞笑。

### 2. Rule of Three

三次重复中，前两次建立模式，第三次打破模式。

```
第一次: 建立预期
第二次: 确认预期
第三次: 打破预期 = 笑点
```

**Example**:
- 第一次加班：默默接受
- 第二次加班：有点烦躁但还是做了
- 第三次加班：直接在公司打地铺，带上了帐篷

### 3. Setup-Punch Misdirection

引导观众朝一个方向思考，然后突然揭示完全不同的现实。

```
Setup: 建立一个预期的方向
    ↓
观众以为: 会朝这个方向发展
    ↓
Punch: 完全出乎意料的结果
```

**Example**:
- Setup: 男朋友精心准备了惊喜
- Expected: 浪漫的告白
- Punch: 女朋友已经知道了一切，反过来给了他更大的惊喜

### 4. "What Could Make It Worse?" Escalation

每次冲突后问这个问题——然后让那个"更糟"的事真的发生。

```
情况A已经很糟了
    ↓
问: 还能更糟吗？
    ↓
情况B发生了——确实更糟了
    ↓
问: 还能更更糟吗？
    ↓
情况C发生了——糟到荒谬了 = 笑点
```

**Key**: 升级必须有**因果逻辑**，不能突然天降灾祸。每一步"更糟"都应该是前一步的自然结果。

### 5. Callback / Running Gag

前面埋下的笑话元素，在关键时刻回响。一次 setup，多次 payoff。

```
Act 1: 埋下笑话种子（一个小细节/口头禅/习惯）
Act 2: 自然地再次出现，观众会心一笑
Act 3: 在最不合适的时候再次出现 = 最大笑点
```

**Example**:
- 妈妈每次打电话第一句话都是"吃了吗"
- 在最严肃的争吵中，妈妈脱口而出"你吃饭了吗？"
- 严肃瞬间被打破，同时揭示了妈妈的本质

### 6. "Yes, and..." Principle

接受荒谬的前提并**加码**，永远不否认、不解释、不合理化。

```
A: 这个外卖员其实是CEO
B(Yes): 对，他每个周末都送外卖
B(And): 而且他的外卖评分比公司股价还重要
```

**Anti-pattern**: "Yes, but..." 会让喜剧氛围降温。

### 7. Enter Late, Exit Early

跳过所有铺垫，从冲突中间开始；在笑点达到最高点时退出。

```
❌ 错误: 铺垫 → 铺垫 → 铺垫 → 铺垫 → 冲突 → 笑点
✅ 正确: 冲突进行中 → 立即升级 → 笑点 → 退出
```

**The first 3 seconds rule**: 喜剧内容的 hook 必须在前 3 秒内让观众意识到"这是搞笑的"。

## Dialogue Techniques

### Subtext in Comedy

喜剧对话的潜台词和正话之间的**落差**就是笑点。

| Surface Text | Subtext | Why Funny |
|-------------|---------|----------|
| "我没事" | 快哭了 | 嘴硬 = 可爱 |
| "随便" | 精心挑选了一小时 | 反差 |
| "你开心就好" | 我很不开心 | 假装大度 |

### The Unspoken Reaction Shot

在喜剧中，**旁观者的反应**和主角的动作一样重要。

```
动作: 主角做了尴尬的事
    ↓
反应: 旁观者的表情——这才是笑点
    ↓
"确认"效应: 观众通过旁观者的反应确认"这确实很好笑"
```

**Without reaction = unwitnessed feat = lacks weight.**

## Comedy Pacing

| Element | Timing | Why |
|---------|--------|-----|
| Setup | 充足时间 | 观众需要理解前提才能笑 |
| Punchline | 精确到秒 | 早一秒或晚一秒都不好笑 |
| Reaction shot | 在punchline之后 | 给观众"确认笑声"的时间 |
| Beat (pause) | 1-2秒 | 让笑声有余地呼吸 |
| Next beat | 不要太急 | 等笑声开始落下再继续 |

**Comedy rhythm = Setup(time) + Punch(instant) + Reaction(confirm) + Beat(breathe)**

## Anti-Patterns (What Kills Comedy)

| ❌ Don't | ✅ Do | Why |
|---------|-------|-----|
| 角色知道自己很可笑 | 角色完全认真 | 自嘲≠喜剧，真诚才是喜剧引擎 |
| 解释笑话 | 让观众自己悟到笑点 | 解释 = 承认观众不懂 = 不再好笑 |
| 笑点太依赖文字 | 用画面+表情+动作 | 视觉笑点比语言笑点更有冲击力 |
| 连续密集笑点 | 笑点之间留呼吸空间 | 没有间隔=疲劳=不好笑 |
| 用"所以呢？"替代笑点 | 每个beats都有明确 payoff | 无payoff的setup=浪费观众时间 |
