# Comic Typography — 漫画文字排版指南

> How to place text in AI-generated comic images for Douyin short-form videos.

## Core Problem

当前工作流把文字当作"视频字幕"——放在画面底部。但漫画的文字是画面的一部分，
应该融在场景里。漫画视频需要多样化的文字位置和风格。

## 六种文字类型

### 1. 钩子标题 (Hook Title)
- **位置:** 画面中央偏上，占据顶部 40% 的安全区
- **时机:** 前 3 秒，第一帧就要出现
- **字体:** 粗体黑体或手写体，60-80px at 1080p
- **动画:** scale pop（0.3s 弹入）或 fade in
- **示例:** "我妈偷偷进了我的高中同学群"
- **Seedream 提示词:** `画面中央上方用粗中文写着"XXXX"，如电影海报标题，白色粗体字带深色描边`

### 2. 对话气泡 (Speech Bubble)
- **位置:** 靠近说话角色的嘴巴附近，用引导线连接
- **字体:** 中等粗细，跟角色绑定（不同角色可以不同字体颜色）
- **气泡样式:** 圆角矩形，白色底+角色色描边，尖角指向说话者
- **字数限制:** 每个气泡最多 15 个字
- **Seedream 提示词:** `角色旁边有对话气泡，气泡内写着"XXXX"，白色圆角气泡`

### 3. 内心独白 (Inner Monologue / OS)
- **位置:** 画面角落（左上或右下），小号字
- **字体:** 倾斜的细体或手写体，比正文小
- **样式:** 可以用括号包裹 `（...）` 或 thought bubble（云朵形气泡）
- **颜色:** 灰色或角色色，半透明
- **Seedream 提示词:** `画面右下角有倾斜的小字"（XXXX）"，灰色手写体`

### 4. 旁白 (Narration)
- **位置:** 画面底部横条，或顶部横条（不要和钩子标题冲突）
- **字体:** 规矩的无衬线体（如思源黑体），42-48px
- **样式:** 半透明黑色背景条（75% opacity），白色文字
- **字数限制:** 每条 20 字以内
- **Seedream 提示词:** `画面底部有横条文字"XXXX"，白色字在深色半透明背景上`

### 5. 情绪强调 (Emphasis Text)
- **位置:** 跟随画面元素，可以扭曲/放大
- **字体:** 夸张的显示字体（如爆炸字、颤抖字）
- **大小:** 比正文大 2-3 倍
- **颜色:** 红色、黄色等高对比色
- **动画:** 放大抖动效果
- **示例:** "炸了！" "完了..." "三年！"
- **Seedream 提示词:** `画面中有一个巨大的红色文字"XXXX"，漫画风格的爆炸字效果`

### 6. IP 结尾 (Brand Outro)
- **位置:** 固定位置——底部居中或画面右下角
- **字体:** 品牌字体（固定不变）
- **内容:** 博主名 + 关注引导
- **样式:** 和正片有区分（不同背景色或边框）
- **详见:** `skills/creative/personal-ip.md`

## 文字位置黄金法则

| 法则 | 说明 |
|------|------|
| 前 3 秒必须有大字 | 80% 用户在静音观看，文字是第一信息 |
| 文字不要挡住表情 | 对话气泡放在角色头部上方或侧面 |
| 同类文字位置一致 | 所有旁白都在底部，所有内心OS都在角落 |
| 每帧最多一个文字元素 | 避免信息过载 |
| 文字颜色和背景对比 4.5:1+ | 白字深底或深字浅底，永远不要在花哨背景上放文字 |

## Seedream 中直接画文字

由于当前工作流直接在 AI 生图时添加文字（而非后期叠加），
需要在 prompt 中明确描述文字位置和样式：

```
好的 prompt 示例：
"一个中年女性在沙发上玩手机，画面中央上方用粗中文写着'我妈偷偷进了我的高中同学群'，
白色粗体字带黑色描边，如电影海报标题"
```

```
不好的 prompt 示例：
"一个中年女性在沙发上玩手机"（缺少文字描述）
```

## 推荐中文字体（后期叠加时使用）

| 用途 | 字体 | 获取方式 |

| 钩子标题 | 思源黑体 Heavy / 站酷快乐体 | Google Fonts / 站酷免费 |

## Applying to OpenMontage

在 Remotion 合成阶段，可以通过不同 scene type 实现多种文字位置：
- `text_card` → 钩子标题（居中大字）
- `callout` → 情绪强调（指向画面元素的标注）
- 新增 `speech_bubble` → 对话气泡（可定位到角色附近）
- 新增 `corner_text` → 内心 OS（角落小字）

对于不经过 Remotion 的简单拼接流程，在 Seedream prompt 中直接描述文字位置和样式。
