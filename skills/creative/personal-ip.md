# Personal IP Strategy — 抖音漫画博主个人品牌打造

> How to build a memorable personal brand as a comic creator on Douyin.

## Why IP Matters

没有个人 IP 的漫画就像匿名投稿——观众看完就划走，不会关注你。
个人 IP 让观众记住"是谁在讲故事"，把内容消费变成关系消费。

## 三要素模型

```
个人 IP = 固定形象 + 固定结尾 + 签名元素
```

### 1. 固定形象 (Brand Avatar)

设计一个代表你的漫画角色，作为博主的"化身"：

**设计原则：**
- **简单** — 越简单越好画，每期都出现，复杂的设计是制作负担
- **辨识度高** — 一个标志性特征就够了（如特定发型、帽子、眼镜）
- **和正片风格区分** — 可以用不同画风（如黑白线稿 vs 彩色正片）
- **有亲和力** — 观众要对这个角色有好感

**固定形象 Prompt 模板：**

```
[你的角色描述]，简笔画风格，黑白线稿，只有一个标志性配色元素，
表情亲和可爱，半身像，纯白背景，作为漫画博主的个人形象
```

**建议：** 只需要 1-2 张固定的角色图，所有视频复用。

### 2. 固定结尾格式 (Outro Template)

每期视频最后 3-5 秒的统一模板：

```
[最后 3-5 秒模板]
┌────────────────────────────────┐
│                                │
│   [你的形象从画面边缘走入/弹出]   │
│                                │
│   "我是[你的名字]，画生活里      │
│    那些被忽略的温柔"             │
│                                │
│   👤 关注我，解锁更多漫画故事    │
│                                │
│   [品牌色边框或角标水印]          │
└────────────────────────────────┘
```

**结尾帧 Seedream Prompt（固定）：**

```
[你的角色]，从画面右侧走入，面带微笑，画面左侧有文字区域留白，
底部有"关注我"图标，简洁干净的背景，[品牌色]点缀
```

**固定元素：**
- 同一个角色（从 Step 1 的固定形象）
- 同一句话签名语（不要每期换）
- 同一个关注引导布局
- 同一个品牌色

### 3. 签名元素 (Signature Elements)

选择 1-2 个"观众一看到就知道是你"的元素：

| 元素类型 | 示例 |
|----------|------|
| 口头禅/签名语 | "我是XX，画生活里那些被忽略的温柔" |
| 标志性动作 | 角色每次都从同一个方向走入 |
| 品牌色 | 每期结尾都用同一种颜色的边框/角标 |
| 音乐 | 同一首 BGM 的最后 3 秒（标志性旋律） |
| 转场动画 | 固定的结尾转场效果 |

## 和 Story Factory 的集成

在 Story Factory 的 5 拍结构中，第 5 拍 (RESOLVE, 80-100%) 自动预留 IP 结尾空间：

```
Beat 5 (48-60s):
  - 前 2 秒: 故事的情感收尾（最后一个画面）
  - 后 3-5 秒: IP 结尾帧（固定模板，每期复用）
```

**建议：** 把 IP 结尾帧预先生成为 1-2 张固定图片，
所有视频在最后拼接相同的 IP 结尾片段。

## 实施步骤

1. **设计你的形象** — 用 Seedream 生成 2-3 个版本的角色设计，选最满意的
2. **确定签名语** — 一句话，不超过 20 字，要有你的个性
3. **选择品牌色** — 一个主色，和一个辅助色
4. **生成结尾帧模板** — 2 张固定图片（一个签名语版本 + 一个关注引导版本）
5. **制作结尾视频片段** — 3-5 秒的固定 MP4，所有视频复用
6. **每期执行** — 正片视频 + 固定 IP 结尾片段 = 最终视频
