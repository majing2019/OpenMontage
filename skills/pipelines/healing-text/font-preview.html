🔧 一键安装所有可下载字体（Homebrew）

brew install font-noto-serif-sc font-noto-sans-sc font-lxgw-wenkai font-ma-shan-zheng font-zcool-kuaile font-zcool-xiaowei

如果某个 brew cask 不存在，手动去 Google Fonts 下载 .ttf 放到 ~/Library/Fonts/ 即可。
安装后重新打开浏览器刷新此页面即可看到本地渲染效果。
